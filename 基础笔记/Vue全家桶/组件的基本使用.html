# 第十八节 组件

# 2.1基本使用

# 2.1.1 Vue中使用组件的三大步骤：

一、定义组件(创建组件)

二、注册组件

三、使用组件(写组件标签)

一、如何定义一个组件？

  • 使用Vue.extend(options)创建，其中options和new Vue(options)时传入的那个options几乎一样，但也有点区别；

​ 区别如下：

  • 1.el不要写，为什么？ ——— 最终所有的组件都要经过一个vm的管理，由vm中的el决定服务哪个容器。
  • 2.data必须写成函数，为什么？ 如果data写成对象的形式，在组件进行复用的时候，多个组件对这一个data存在引用关系，改变其中一个组件将会对其他组件里的data也会改变，如果写成函数的形式，使用data将会返回一个全新的值———— 也就是避免组件被复用时，数据存在引用关系。
  • 备注：使用template可以配置组件结构。

二、如何注册组件？

  • 1.局部注册：靠new Vue的时候传入components选项
  • 2.全局注册：靠Vue.component('组件名',组件)

三、编写组件标签：

# 2.1.2 具体创建组件步骤：

第一步：创建school组件，其中要包含school的信息

const s = Vue.extend({
  date(){
    return {
      schoolName:"ax",
      addtress:"ab"
    }
  },
  methods:{
    showName(){
      alert(this.schoolName);
    }
  }
})

第二步：注册组件（全局注册和局部注册）创建Vue实例vm选择配置的标签并管理组件

var vm = new Vue({
  el = "root",
  data:{
	msg:"hello"
  }
  // 第二步：注册组件（局部注册）
  components:{
  	school:s          
  }
})
// 全局注册要写在vue实例外面，通过component方法
Vue.component('hello',hello)

第三步：使用组件，通过在html里直接编写组件标签进行使用，而组件标签名称就是组件的名称

<div id="root">
			<hello></hello>
			<hr>
			<h1>{{msg}}</h1>
			<hr>
			<!-- 第三步：编写组件标签 -->
			<school></school>
		</div>

# 2.1.3 几个注意点

  • 1、关于组件名：

  • 一个单词时可开头大小写

  • 多个单词组成时

    ​ （1）(kebab-case命名)：my-school

    ​ （2）第二种写法(CamelCase命名)：MySchool (需要Vue脚手架支持，不然会报错)

  • 组件名尽可能回避HTML中已有的元素名称

  • 2、关于组件标签

  • 可写成 和，但是不用使用脚手架时，会导致后续组件不能渲染。

  • 3、简写

  • const school = Vue.extend(options) 可简写为：const school = options

# 2.2 组件的嵌套

  • 组件的嵌套可以理解为在创建新的组件时，在新组件内部注册创建过的组件，最后在vue实例中注册这个新的组件即可
  • 代码演示
	//定义school组件
		const school = Vue.extend({
			name:'school',
			template:`
				<div>
					<h2>学校名称：{{name}}</h2>	
					<h2>学校地址：{{address}}</h2>	
					<student></student>
				</div>
			`,
			data(){
				return {
					name:'ax',
					address:'ab'
				}
			},
			//注册组件（局部）
			components:{
				student
			}
		})

		//定义hello组件
		const hello = Vue.extend({
			template:`<h1>{{msg}}</h1>`,
			data(){
				return {
					msg:'axab！'
				}
			}
		})
		
		//定义app组件
		const app = Vue.extend({
			template:`
				<div>	
					<hello></hello>
					<school></school>
				</div>
			`,
			components:{
				school,
				hello
			}
		})

		//创建vm
		new Vue({
			template:'<app></app>',
			el:'#root',
			//注册组件（局部）
			components:{app}
		})

# 2.3 关于VueComponent

1、school组件的本质是一个名为VueComponent的构造函数，且不是程序员定义的，是Vue.extend生成的。

2.我们只需要写或，Vue解析时会帮我们创建school组件的实例对象，

即Vue帮我们执行的是：

new VueComponent(options)。

3.特别注意：每次调用Vue.extend，返回的都是一个全新的VueComponent！！！！

4.关于this指向：

(1).组件配置中：

​ data函数、methods中的函数、watch中的函数、computed中的函数 它们的this均是VueComponent实例对象。

(2).new Vue(options)配置中：

​ data函数、methods中的函数、watch中的函数、computed中的函数 它们的this均是Vue实例对象。

5.VueComponent的实例对象，以后简称vc（也可称之为：组件实例对象）。

​ Vue的实例对象，以后简称vm。

# 2.5单文件组件

  • 文件后缀名为 .vue
  • 第一小节-------先了解.vue文件里该怎么写
<template>
    // 组件的结构
	<div class="demo">
		<h2>学校名称：{{name}}</h2>
		<h2>学校地址：{{address}}</h2>
		<button @click="showName">点我提示学校名</button>	
	</div>
</template>

<script>
	// 组件交互相关的代码（数据、方法等等）
    export default { // ES6模块化，让他暴露，外部通过import引入
	// 三种暴露方式
		name:'School',
		data(){
			return {
				name:'ax',
				address:'ab'
			}
		},
		methods: {
			showName(){
				alert(this.name)
			}
		},
	}
</script>

<style>
	.demo{
		background-color: orange;
	}
</style>
  • app.vue : 汇总所有的组件
<template>
	<div>
		<School></School>
		<Student></Student>
	// 最后再使用
	</div>
</template>

<script>
	//引入组件
	import School from './School.vue'
	import Student from './Student.vue'

	export default {
		name:'App',
		components:{ // 光引用还不行，还要注册
			School,
			Student
		}
	}
</script>
  • main.js : 创建VUE实例
  • app.vue才配与main.js对话
import App from './App.vue'

new Vue({
	el:'#root',
	template:`<App></App>`,
	components:{App},
})
  • index.html ：为容器
<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8" />
		<title>练习一下单文件组件的语法</title>
	</head>
	<body>
		<!-- 准备一个容器 -->
		<div id="root"></div>
		<script type="text/javascript" src="../js/vue.js"></script>
		<script type="text/javascript" src="./main.js"></script>
        // 引用这两句话意义不大，因为在这是不能运行的
	</body>
</html>
